# 慕课网一个热门的vue教程---饿了么外卖app

> vue2.0、vuex、vue-router、axios、webpack、eslint、better-scroll

### 安装步骤


```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
